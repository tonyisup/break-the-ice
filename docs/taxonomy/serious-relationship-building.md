# Serious Relationship-Building Taxonomy

## Purpose

This taxonomy expands Break the Ice beyond playful icebreakers with question mechanics and language registers designed for voluntary, substantive relationship building. The goal is to help people understand one another more clearly—not to manufacture vulnerability.

## Design boundary

A serious question is not automatically an intimate or useful question. These entries are designed around four constraints:

1. **Specific before abstract:** Ask for a moment, behavior, preference, or bounded hope rather than a totalizing identity statement.
2. **Voluntary depth:** Every prompt must support a meaningful light answer as well as a deeper answer.
3. **No diagnosis or extraction:** Do not use therapy labels, presume pain, ask for secrets, or treat disclosure as proof of closeness.
4. **Relational yield:** The answer should give another person something concrete to understand, remember, or respond to.

## Added tones

| Slug       | Name     | Register                                   | Seriousness | Intimacy |
| ---------- | -------- | ------------------------------------------ | ----------- | -------- |
| `candid`   | Candid   | Plainspoken, honest, direct, nonjudgmental | 5/5         | 5/5      |
| `tender`   | Tender   | Warm, careful, emotionally attentive       | 4/5         | 5/5      |
| `grounded` | Grounded | Steady, mature, concrete, unforced         | 5/5         | 4/5      |

## Pairing guidance

- **Formative Moment + Grounded:** Best default for substantial but socially manageable stories.
- **Values in Practice + Candid:** Good for learning what someone actually prioritizes without asking for a values résumé.
- **How to Know Me + Tender:** Highest intimacy pairing; use when the audience has enough trust for relational preferences.
- **Future Chapter + Grounded:** Useful for hopes and direction without career-interview language.
- **How to Know Me + Candid:** Direct and high-intimacy; should remain optional and never be used as a forced group round.

## Safety guidance

These entries are marked `medium` risk because the same well-phrased prompt can land differently depending on trust, hierarchy, and setting. Product surfaces should not imply that a person must answer deeply. Avoid using the highest-intimacy pairings in mandatory workplace contexts or with mixed power levels unless participants can skip without penalty.
